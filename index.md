# Chlorine

Chlorine是一款适用于“语言不精确输入”的应用，其目的在于改善搜索引擎中由于不规范中文导致结果偏离的问题。

目前Chlorine可以在PyPI中安装，提供了whl文件和.tar.gz的安装包。

```powershell
pip install chlorine
```
